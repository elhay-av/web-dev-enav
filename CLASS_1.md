# שיעור 1 - מבוא


## היכרות
- איך תוכנה וקוד משפיעים על עלינו?
- מה אנחנו כבר יודעים?
- מה זה אתר אינטרנט?


## מה נעשה בקורס
- נלמד מה זה קוד ואיך כותבים קוד
- נלמד איך בנויים אתרי אינטרנט
- נבין איך תקשורת בין מחשבים עובדת
- כל אחד יכין אתר/אפליקציה אישי


## לעבודה
- נכתוב קוד פעם ראשונה ונראה מה קורה


``` html
<h1>שלום לכולם</h1>
```

``` html

```
